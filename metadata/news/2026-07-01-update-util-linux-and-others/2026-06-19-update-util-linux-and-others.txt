There is a serious issue with sys-apps/util-linux for split-usr.  The live
ebuild was carefully not released with split-usr adjustments by the distro and
not well tested for this use case.

IMPORTANT:  If you are merge-usr or live systemd user, you do not need to worry.
You do not need to follow these steps.

IMPORTANT:  If you are a split-usr and haven't rebooted, you need to sync
update the repo (emaint sync -A) and emerge -1vO sys-apps/util-linux immediately
with the split-usr fix to avoid increasing the remediation cost.

To fix the issue, you need to use rescuecd like SystemRescue (formerly
SystemRescueCd).  If you have an updated WiFi device, it is strongly recommended
to update the latest SystemRescue for driver compatibility.

Step 1:  mount and chroot

1.  Boot into system with SystemRescue CD.

2.  Decrypt the device

3.  mkdir /mnt/gentoo

4.  Log in the WiFi

5.  Copy /etc/resolv.conf to /mnt/gentoo/etc

6.  Run the mount script:
Contents of rescue-mount.sh:
#!/bin/bash
mount --types proc /proc /mnt/gentoo/proc
mount --rbind /sys /mnt/gentoo/sys
mount --make-rslave /mnt/gentoo/sys
mount --rbind /dev /mnt/gentoo/dev
mount --make-rslave /mnt/gentoo/dev
mount --bind /run /mnt/gentoo/run
mount --make-slave /mnt/gentoo/run

7. Chroot:  chroot /mnt/gentoo /bin/bash

Step 3:  Generate the Manifest

# When you are done with your local copy, do the following:
ebuild util-linux-9999.ebuild digest

Step 4:  Emerge the changes:
EVCS_OFFLINE=1 emerge -1vO =util-linux-9999

Okay that fixes util-linux for split-usr profiles.  Now you need to fix 2 more
packages -- firejail and xinit any package that depends on util-linux.

Step 5:  Uninstall and install firejail
emerge -C firejail
EVCS_OFFLINE=1 emerge -1vO firejail

This is necessary to fix the wrappers generated by firejail for util-linux if any.

Step 6:  Reinstall xinit

emerge -1vO xinit

This will fix the autodetected configure time paths.

This should fully restore back your system.


Alternative route:

Boot using system SystemRescue.  Then perform the stage 3 tarball.  This is not
recommended for experienced users.  For new users, this is another option.
