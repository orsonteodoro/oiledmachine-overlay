If the following error is encountered:

Error: too many environment variables: >= env-max-count (256)

Either set env-max-count 512 in /etc/portage/firejail.config or
`emerge -vO firejail`.

The 512 should be used when emerging, but downgraded back to 256 when using apps
after install.  The ebuild or build system adds too many of them for the default
to be used.
